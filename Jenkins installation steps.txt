Jenkins Pre-Requisite: Java installation:
=========================================

1) sudo add-apt-repository ppa:webupd8team/java
2) sudo apt-get update
3) sudo apt install openjdk-8-jdk
4) java -version

Jenkins Installation:
=====================


1) wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -

2) sudo sh -c 'echo deb https://pkg.jenkins.io/debian binary/ > /etc/apt/sources.list.d/jenkins.list'

3) sudo apt-get update

4) sudo apt-get install jenkins
